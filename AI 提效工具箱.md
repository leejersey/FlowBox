## 🧰 AI 提效工具箱（桌面应用）

> 目标：做成一个本地桌面应用（Electron / Tauri），开机即用，不依赖浏览器，深度集成 AI 能力。
> 品牌定位：**"FlowBox"桌面端** — 比懂王更懂 AI 的效率利器。

### 📦 V1 基础功能模块

| 模块 | 功能 | 核心体验 |
|------|------|----------|
| 🎙️ 语音记录 | 录音→转文字→AI 提取待办/摘要 | 随便说一段话，AI自动拆出To-Do加入列表 |
| 📋 剪贴板管家 | 图片/文本/代码片段历史记录 | 一键回溯，再也不怕覆盖 |
| ⏱️ 工作时间统计 | 自动记录各应用使用时长 | 可视化饼图/柱状图，看清时间都去哪了 |
| ✅ Todolist | 待办管理 + 优先级 + 标签 | 支持从语音/剪贴板自动导入 |
| 💡 Idea 速记 | 快捷键一键弹窗记录灵感 | 零摩擦捕获，不打断任何工作流 |
| 🍅 专注番茄钟 | 番茄工作法 + 专注时长统计 | 可视化专注趋势，量化心流状态 |

### 🤖 AI 增强方案

#### 1. 语音 → AI 智能拆解
- 转文字后自动调用 LLM 提取：待办事项、核心观点、会议纪要
- 自动将 To-Do 写入 Todolist，打上时间和来源标签

#### 2. 剪贴板 → OCR + 智能分类
- 图片静默 OCR（代码截图、UI 设计稿、发票等）
- 自动打标签：`#代码` `#设计` `#文档` `#发票`
- 全文检索所有历史剪贴板内容

#### 3. 剪贴板 → AI 即时处理
- 复制报错代码 → 自动识别 → 旁边弹出"AI 修复"按钮
- 复制英文 → 自动翻译悬浮展示
- 复制长文本 → 一键摘要

#### 4. 专注 + AI 复盘
- 番茄钟结束自动生成"本次专注小结"
- 一天结束 → AI 根据完成的 Todo + 记录的 Idea → 一键生成日报/周报
- 专注趋势分析：哪些时间段效率最高

### 🚀 进阶创意

#### 全局"智能管家"悬浮窗
- `Option + Space` 全局唤醒极简对话框
- 拖入代码/文本 → 解释、翻译、润色公众号文案
- 用完即关，不打断工作流
- 支持上下文记忆，知道你当前在干什么

#### 碎片灵感"连点成线"
- AI 每周自动聚类分析零散 Idea
- 发现关联，碰撞出新选题/产品方向
- 推送"本周灵感周报"

#### 自动化防打扰联动
- 开启番茄钟 → 自动触发 macOS 勿扰模式
- 自动设置微信/飞书状态为"忙碌"
- 屏蔽指定 App 通知，强行进入心流

### 💎 更多创意补充

#### 5. 智能日程感知
- 读取系统日历，番茄钟开始前提醒"15分钟后有会议"
- 根据 Todo 优先级和预估时间，AI 自动建议今日时间块分配
- "你上午效率最高，建议把这个难任务放在 9:00-11:00"

#### 6. 知识碎片 → 自动归档到 Obsidian
- 语音转文字、剪贴板笔记、Idea 按主题自动归入 Obsidian 对应文件夹
- AI 自动打双链 `[[]]`，和现有笔记建立关联
- 相当于一个"自动第二大脑整理员"

#### 7. 代码片段管理 + AI 增强
- 复制的代码自动归档，按语言/项目分类
- AI 自动加注释、补全类型标注
- 支持快捷搜索："上周那段 Python 异步处理代码"

#### 8. 截图 → AI 秒懂
- 全局截图快捷键，截完自动 OCR + AI 分析
- 截 UI 稿 → AI 生成前端代码骨架
- 截报错 → AI 直接给解决方案
- 截表格 → 自动转 Markdown/Excel

#### 9. 智能提醒与习惯追踪
- 久坐提醒 + 喝水提醒（可自定义间隔）
- 记录每日专注时长、完成任务数，生成"效率成长曲线"
- 连续专注天数打卡，游戏化激励

#### 10. "智能管家"快捷指令系统
- 自定义 Prompt 模板绑定快捷键
- 例：`Cmd+Shift+R` → 选中文本一键润色
- 例：`Cmd+Shift+T` → 选中代码一键写单测
- 可分享指令模板，社区化

### 🛠️ 技术选型建议

| 层级 | 方案 | 理由 |
|------|------|------|
| 桌面框架 | **Tauri 2.0**（首选）/ Electron | Tauri 更轻量，打包仅 ~5MB |
| 前端 | React + TypeScript | 你已有 React 经验 |
| 本地存储 | SQLite（better-sqlite3） | 轻量嵌入式，无需服务端 |
| AI 接入 | DeepSeek / 通义千问 API | 国内模型，速度快成本低 |
| 语音转文字 | Whisper（本地）/ 阿里 ASR | 隐私敏感选本地，否则云端 |
| OCR | PaddleOCR（本地）/ 阿里 OCR | 中文识别精度高 |
| 全局快捷键 | Tauri globalShortcut API | 原生支持 |
| 系统联动 | macOS AppleScript / shell | 勿扰模式、应用控制 |
